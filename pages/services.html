Remote Assistance & On Field

Project Overview

Establishing a structured remote assistance system for managing IT and computer-related issues, ensuring efficient query resolution, ticket tracking, and escalation.

Communication & Query Collection

Students, staff, and lab users submit queries via a dedicated WhatsApp Group. An auto-response system acknowledges receipt, and urgent queries can be flagged.

  • WhatsApp Group for query submission
  • Automated acknowledgment system
  • Urgent query flagging

Technicians' Interactive Logging System

A simple web-based system for technicians to log incoming issues with detailed fields for user's name, issue description, device ID, and priority level.

  • User's Name
  • Issue Description
  • Device ID / Lab Number
  • Priority Level (Urgent, Critical, High, Violation)

Issue Tracking & Escalation

A central platform for real-time monitoring of ticket status. Unresolved issues are reassigned or escalated within 48 hours to ensure timely resolution.

  • Real-time ticket status monitoring
  • Automated reassignment for unresolved issues
  • Escalation to higher levels if needed

Technician Roles & Responsibilities

Dedicated technicians (O and T) handle tickets based on workload. An escalation framework ensures issues are addressed even if a primary technician is unavailable.

  • Workload-based ticket distribution
  • Three-level escalation framework: Assigned Tech → Another Tech → IT Head/External Support

Procurement & Maintenance

Streamlined processes for handling new hardware requirements (reported to management) and logging/assigning hardware maintenance tasks.

  • Management approval for procurement
  • Logged and assigned hardware maintenance

Task Time Tracking & Reporting

The system records ticket open and close times to measure resolution efficiency. Weekly reports provide insights into response times and issue trends.

  • Resolution time tracking
  • Weekly efficiency reports
  • Analysis of unresolved issues and trends

WhatsApp Auto-Response & Assurance

Automated messages for standard acknowledgment, status updates, and escalation notices, keeping users informed throughout the resolution process.

  • "Issue logged" acknowledgment
  • "Actively working" status updates
  • "Issue escalated" notifications
